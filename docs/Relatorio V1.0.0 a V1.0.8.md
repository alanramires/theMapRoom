# Relatorio Consolidado - v1.0.0 a v1.0.8

## Visao geral
Fase de formacao do jogo: base principal, combate inicial e fechamento do primeiro ciclo de embarque.

## Destaques do periodo
1. Base do projeto (v1.0.0 a v1.0.3)
- mapa hex, turno, dados e ferramentas iniciais.

2. Combate inicial robusto (v1.0.4)
- sensor + calculo + simulacao para balanceamento.

3. Mobilidade mais tatica (v1.0.6)
- bonus de estrada e ajustes de camada.

4. Embarque de ponta a ponta (v1.0.7 e v1.0.8)
- validação, confirmacao e execucao completa.

## Bloco tecnico curto
- Consolidacao do `TurnStateManager`.
- Fortalecimento de sensores principais.
- Primeira grande rodada de tooling para iteracao rapida.
