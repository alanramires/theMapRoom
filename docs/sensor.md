# Atalhos de Sensores

Atalhos disponíveis após mover a unidade:
(estados: Moveu Andando / Moveu Parado)

- `A` - Aim (Mirar)
- `E` - Embark (Embarcar)
- `D` - Disembark (Desembarcar)
- `C` - Capture (Capturar)
- `F` - Merge (Fundir)
- `S` - Supply (Suprir)
- `T` - Transfer (Transferir)
- `L` - Layer / Mudar Altitude
- `M` - Move Only (Apenas mover/finalizar sem sensor)
- `ESC` - Desfazer/voltar no fluxo atual

## Observacao

- Os atalhos acima aparecem conforme disponibilidade do sensor no contexto da unidade/celula.
