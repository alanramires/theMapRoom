# Relatorio de Atualizacao - v1.0.0 a v1.0.3

## Em uma frase
Foi a fundacao do projeto: mapa hex, turno, movimento e base de dados inicial.

## O que isso trouxe na pratica
- O jogo deixou de ser experimento e ganhou estrutura jogavel.
- Ficou possivel evoluir sistemas sem recomeçar do zero.
- Comecou a base que sustentou combate, logistica e operacoes especiais depois.

## Bloco tecnico curto
- Estrutura inicial de `TurnStateManager`, cursor e regras de ocupacao.
- Primeiros bancos: unidades, terrenos, construcoes e estruturas.
- Spawners e ferramentas iniciais de editor para acelerar testes.
